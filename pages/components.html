Slate

Boxes

Currently the box class is the block class and vice versa. The idea for blocks did not come until later and at the time blocks was the name of this component. However, I thought that blocks would better fit the new idea and once completed the classes will switch names. Boxes are be simple rectangular containers to put elements in.

class="block"

Soon to be changed to "box"

class="block center-text"

Like many container elements in slate, the "center-text" class can be used to center text elements within the box.

class="block fit-content"

Adding the "fit-content" class will re-size the box so that it is only as wide as needed to fit the content.

class="block fit-content center-text"

Here it is all together.

Blocks

As previously mentioned this class is actually the box class. Once completed it will be switched to be the block class. When complete blocks will be similar to boxes except with flex-box capabilities.
