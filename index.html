Slate


Introduction

Slate is simply a CSS "framework" that was made as a side project so that I could better lean how to use CSS. It is largely based off of the Bulma and Bootstrap frameworks.


Getting Started

At this point Slate is simply a personal project. The creation of this website is mostly used for testing. However, who knows maybe someday it will get some actual use.


Overview

The point of Slate was to better learn CSS while also creating a simple framework that I could use as an alternative to bootstrap or buma in scenarios where perhaps due to project requirements couldn't use a widely used framework. The focus of Slate is to cut down on the ammount of CSS and styling work one has to put into a website by as much as possible. In the future I hope to incooperate JS elements similar to Semantic UI so that Slate can even handle more functional and advanced styling such as the toggleable menu elements on the side bar of this page.
